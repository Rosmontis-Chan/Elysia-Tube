## âœ¨ Deskripsi Proyek

**Elysia Tube** adalah proyek *client* YouTube minimalis yang seluruhnya dibangun dalam **satu file (`index.html`)**. Proyek ini menampilkan video dari YouTube menggunakan YouTube Data API v3 dengan desain *soft*, *glowing*, dan cerah yang didominasi warna pink (Elysia) dan putih.

Tujuannya adalah menciptakan pengalaman menonton yang ringan dan cepat, sekaligus sebagai contoh yang *cute* untuk implementasi API dasar *frontend*.

## ðŸŒŸ Fitur Utama

* **Desain *Glowing* & Lembut:** Antarmuka dengan tema Pink-Putih yang elegan, halus, dan *glowing*.
* **Satu File *Magic***: Seluruh HTML, CSS, dan JavaScript berada dalam `index.html`.
* **Pencarian Video Instan:** Mencari video YouTube secara real-time menggunakan YouTube Data API.
* **Informasi Lengkap:** Menampilkan judul dan **Nama Channel** pada setiap kartu video.
* **Animasi *Intro* Cantik:** Dilengkapi *preloader* *glowing* **"Elysia Tube"** saat halaman pertama kali dibuka.
* **Pemutar Video Bersih:** Memutar video dalam *pop-up* minimalis dengan efek *loading* **Bunga Sakura** saat diklik.

## ðŸš€ Cara Penggunaan

Ikuti langkah-langkah mudah ini untuk menjalankan Elysia Tube di lokal Anda:

### 1. Dapatkan File Proyek

1.  Klik tombol **"Code"** (atau "Download ZIP") di halaman repositori ini.
2.  Ekstrak file **`index.html`** di komputer Anda.

### 2. Dapatkan API Key

Anda memerlukan **YouTube Data API Key** dari Google Cloud Console.

### 3. Masukkan Kunci API

1.  Buka file `index.html` menggunakan *text editor* (seperti VS Code atau Notepad++).
2.  Cari baris berikut di dalam tag `<script>` (di bagian bawah):
    ```javascript
    const API_KEY = 'YOUR_API_KEY_HERE';
    ```
3.  Ganti `'YOUR_API_KEY_HERE'` dengan kunci API Anda yang sebenarnya.

### 4. Jalankan

Buka file **`index.html`** langsung di *browser* Anda (klik dua kali pada file tersebut) dan *website* siap digunakan!
